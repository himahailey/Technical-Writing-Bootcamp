# Feature Announcement: Dark Mode

## Overview

We are excited to introduce Dark Mode, a new feature designed to improve user comfort and reduce eye strain in low-light environments.

## What's New

Dark Mode provides:

* A darker color scheme across the application
* Improved visibility in low-light conditions
* Reduced screen glare
* Consistent appearance across supported devices

## How to Enable Dark Mode

1. Open the application settings.
2. Select **Appearance**.
3. Choose **Dark Mode**.
4. Save your changes.

The application will immediately switch to the dark theme.

## Benefits

Dark Mode helps users:

* Work comfortably for longer periods
* Reduce eye fatigue
* Improve readability in dim environments

## Availability

Dark Mode is available starting with version 3.0.
